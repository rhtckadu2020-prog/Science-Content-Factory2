මුලින්ම, **කාබෝහයිඩ්රේට** (Carbohydrates). ... මේවා කාබන්, හයිඩ්‍රජන් සහ ඔක්සිජන් වලින් හැදිලා තියෙනවා. ... විශේෂයෙන්ම හයිඩ්‍රජන් සහ ඔක්සිජන් පවතින්නේ දෙකට එකක් (2:1) කියන අනුපාතයටයි. ... ශාකවල ප්‍රභාසංශ්ලේෂණයෙන් (Photosynthesis) හැදෙන ප්‍රධාන බලශක්ති ප්‍රභවය තමයි මේවා. ...
කාබෝහයිඩ්රේට ප්‍රධාන වශයෙන් කොටස් තුනකට බෙදෙනවා. ... **මොනොසැකරයිඩ** (Monosaccharides), **ඩයිසැකරයිඩ** (Disaccharides) සහ **පොලිසැකරයිඩ** (Polysaccharides) කියලයි. ...

සරලම සීනි වර්ග තමයි මොනොසැකරයිඩ. ... ග්ලූකෝස් (Glucose) සහ පෘක්ටෝස් (Fructose) මේවාට උදාහරණයි. ... ඩයිසැකරයිඩ කියන්නේ මොනොසැකරයිඩ අණු දෙකක් එකතු වෙලා හැදෙන ඒවා. ... කෑමට ගන්නා සීනි (සුක්‍රෝස් - Sucrose) සහ කිරිවල තියෙන සීනි (ලැක්ටෝස් - Lactose) මේවාට හොඳම උදාහරණයි. ... පොලිසැකරයිඩ කියන්නේ ගොඩක් මොනොසැකරයිඩ එකතු වෙලා හැදෙන සංකීර්ණ කාබෝහයිඩ්රේට. ... බත් වල තියෙන පිෂ්ටය (Starch) සහ ශාක සෛල බිත්තිවල තියෙන සෙලියුලෝස් (Cellulose) මේවාට අයත් වෙනවා. ...

ඊළඟට අපි **ප්‍රෝටීන්** (Proteins) ගැන කතා කරමු. ... ප්‍රෝටීන් කියන්නේ අපේ ශරීරයේ ව්‍යුහය හදන, ... හැම වැඩකටම වගේ උදව් කරන, ... ඇමයිනෝ අම්ල (Amino Acids) වලින් හැදෙන සංකීර්ණ අණු සමූහයක්. ... කෙස්, මාංශ පේශි, සෛල පටල වගේ දේවල් ප්‍රෝටීන් වලින් හැදිලා තියෙනවා. ... ඒ වගේම අපේ ආහාර ජීර්ණ එන්සයිම (Enzymes), හෝර්මෝන (Hormones) සහ ලෙඩ රෝග වලින් ආරක්ෂා කරන ප්‍රතිදේහ (Antibodies) වුණත් ප්‍රෝටීනවලින් තමයි හැදෙන්නේ. ...

දැන් අපි **ලිපිඩ** (Lipids) ගැන බලමු. ... ලිපිඩ කියන්නේ මේද අම්ල (Fatty Acids) සහ ග්ලිසරෝල් (Glycerol) එකතුවෙලා හැදෙන අණු. ... මේවා ජලයේ දියවෙන්නේ නැහැ. ... පොල්තෙල්, බටර් වගේ දේවල් ලිපිඩ වලට උදාහරණයි. ... ලිපිඩ වලින් අපිට ගොඩක් ශක්තිය ලැබෙනවා. ... ඒ වගේම අපේ සෛල පටල හදන්නත්, ... ශරීරය ඇතුළේ තියෙන අවයව ආරක්ෂා කරන්නත්, ... ශරීරයේ උෂ්ණත්වය පාලනය කරන්නත් ලිපිඩ වැදගත් වෙනවා. ...
